# CyberDev Portfolio

A modern, responsive portfolio website for a Full-Stack Developer specializing in web development and cyber systems.

## Features

- Responsive design that works on all devices
- Modern cyber-themed UI with neon accents
- Interactive elements and animations
- Skills section with progress bars
- Projects showcase
- Contact form with validation
- Smooth scrolling navigation

## How to View Locally

1. Open `index.html` in your web browser
2. Or use VS Code Live Server extension

## Deployment to GitHub Pages (Free)

1. Create a GitHub account
2. Create a new repository named `username.github.io`
3. Upload all 4 files to your repository
4. Go to repository Settings > Pages
5. Under "Source", select "main" branch and save
6. Your site will be live at `https://username.github.io`
